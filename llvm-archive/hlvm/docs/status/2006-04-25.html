Status As Of April 25,2006

HLVM is just getting started. The idea was generated from some discussion on the LLVM IRC channel. There is much that needs to be done in order to get LLVM fully coherent as a virtual machine. While LLVM provides the raw materials, HLVM will provide the finished product.

Current, we have been collecting resources. As of today, we have:

  • Four interested developers
  • Reserved hlvm.org domain name
  • Brought up a Subversion repository for the code
  • Built a wiki
  • Built an introductory web site
  • Discussed many ideas about what HLVM should be
  • Documented some architecture and data flow in the wiki
