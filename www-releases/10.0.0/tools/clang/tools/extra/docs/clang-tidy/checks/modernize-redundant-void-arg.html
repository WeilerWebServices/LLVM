modernize-redundant-void-arg¶

Find and remove redundant void argument lists.

Examples:
Initial code Code with applied fixes
int f(void); int f();
int (*f(void))(void); int (*f())();
typedef int (*f_t(void))(void); typedef int (*f_t())();
void (C::*p)(void); void (C::*p)();
C::C(void) {} C::C() {}
C::~C(void) {} C::~C() {}
